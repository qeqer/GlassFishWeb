1) Получение списка театров/получение списка сценариев (С главной на страницу театры/сценарии по ссылке из меню и, возможно, фильтрация с помощью окна поиска)
2) Получение списков шоу по выбранному сценарию и(или) театру и(или) дате проведения (С главной страницы на страницу списка шоу, на которой доступна фильтрация по указанным параметрам)
3) Просмотр занятых работников (сценариев, актеров) на шоу или в театре, возможно с их краткой биографией (С главной страницы на страницу списка театров/шоу, где либо фильтром, либо поиском глазами находится нужный обьект и кликается на него, что перекидывает на конкретную страницу театра/шоу)
4) Получение списка мест на данное шоу (На странице шоу выбирается нужное шоу, на него кликается, появляется отдельная страница, на которой показан в том числе список доступных мест. Есть возможность добавить в корзину, потом внизу страницы, после выбора мест, есть кнопка забронировать выбранные, которая выдает поле, где можно ввести имя/фамилию/id)
5) Покупка билетов (описано в предыдущем пункте)
6) Добавление/удаление/чтение/редактирование чего угодно из вышеперечисленного (редактирование/удаление с помощью отдельной кнопки на странице обьекта, добавление с помощью отдельной страницы, доступной в административном режиме)


Сценарий - произведение, по которому ставится
Шоу - конкретное мероприятие с конкретным местом по конкретному сценарию с конкретной датой прим.меня
Главная страница: краткая информация о сайте, ссылки на (вверху страницы или сбоку):

Страница театров: полный список театров в виде списка с адресом и краткой биографией (Возможность фильтрации вверху страницы по имени, перед списком театров. Переход на конкретный театр с полной информацией по клику (какие шоу идут, кто работает(переход на участников), адрес, биография))
Страница сценариев: список сценариев с поиском конкретного сценария. Кликом на название вызывается отдельная страница для отображения того, что за сценарий, где идет(переход на театр), когда идет (переход на шоу)
Страница списка шоу: возможность выбрать конкретное шоу для просмотра, кто в нем участвует(переход на участников), его дату (поиск по дате, по сценарию, по участникам), перейти на страницу конкретного шоу для выбора мест для этого шоу
Страница участников: отображение всех работников с краткой биографией (фильтрация по шоу или по театру, где работает)

Соответственно требуются еще административные интерфейсы для добавления/удаления вышеперечисленного под что будет отведена еще одна страница, возможно как-то защищенная, на которой будет выбор типа для создания и поля для этого типа, а также кнопка "добавить"
Вход в режим привилегированного пользователя будет осуществляться через страницу(sitename.ru/admin), не отображаемую в меню, по логин/паролю, заданным в конфиг файле

Также будут реализованы процедурно генерируемые странцы для каждого обьекта, с подробной информацией о нем (страница конкретного театра (имя, адрес, биография, список залов, список шоу с возможностью перехода на страницу шоу), страница конкретного шоу (сценарий с возможностью перехода, дата, продолжительность, участвующие актеры с возможностью перехода, театр с возможностью перехода в котором идет, номер зала), страница сценария (с списком шоу по этому сценарию, с описанием и названием))

Также на каждой странице конкретного обьекта в режиме привелигированного пользователя будет доступна кнопка "редактировать", позволяющая редактировать поля, находящиеся на странице.
